# Mental Health Support Chatbot

A web-based mental health support chatbot that provides text-based interaction with optional voice support.  
The system is designed to offer basic emotional support and mental health awareness using AI-driven logic.

> This project is for educational and early-support purposes only and is not a replacement for professional mental healthcare.

---

## Overview

Mental health challenges such as stress, anxiety, and emotional fatigue are increasing, while access to immediate support remains limited.  
This project explores how AI and web technologies can be used to provide a simple, accessible mental health support interface.

The chatbot allows users to:
- Express thoughts through text (and optional voice input)
- Receive empathetic, supportive responses
- Access guidance in a non-judgmental environment

---

## Features

- Text-based chat interface  
- Optional voice input and voice response support  
- NLP-based emotion-aware responses  
- Basic crisis keyword detection with safety messaging  
- Flask-based web application  
- Modular design for future AI upgrades  

---

## Tech Stack

- Language: Python  
- Backend: Flask  
- AI / NLP: Rule-based logic (extendable to ML/DL)  
- Speech (optional):
  - speech_recognition
  - pyttsx3
- Version Control: Git & GitHub
